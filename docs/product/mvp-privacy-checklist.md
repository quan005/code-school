# MVP Privacy Checklist

## Product Surface

- [x] No public student profile pages exist.
- [x] No student search or discovery surface exists.
- [x] No open chat or user-to-user messaging exists.
- [x] No social posting, comments, or direct sharing exists.

## Data Collection

- [x] Child learner state is limited to learner profile, progress, mastery, and submissions.
- [x] Lesson content remains in-repo rather than stored as user-generated content.
- [x] Telemetry is disabled by default.
- [x] Allowed telemetry is limited to coarse product-learning events:
  - lesson status updates
  - practice runs with pass/fail counts

## Data We Do Not Collect For MVP

- [x] Public profile metadata
- [x] Chat messages
- [x] Social graph / friend lists
- [x] Precise location
- [x] Raw reflection-answer analytics
- [x] Raw code snapshots in telemetry streams

## Launch Readiness Checks

- [x] Learner profiles are adult-managed.
- [x] Learner profiles are private by default.
- [x] Route exposure has been reviewed for public student pages.
- [x] Privacy assumptions are documented for engineering.
- [ ] Formal legal policy review is still required before production launch.
