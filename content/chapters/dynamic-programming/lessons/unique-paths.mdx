<ProblemStatement title="How many different routes reach the bottom-right corner?">
  A robot starts in the top-left of an <code>m x n</code> grid and may move only
  right or down. Return the number of unique paths to the bottom-right corner.
</ProblemStatement>

## Example

```txt
m = 3, n = 3
answer = 6
```

<Intuition title="A cell can only be entered from above or from the left">
  Since the robot can only move right or down, each cell’s path count is the sum
  of the path count above it and the path count to its left.
</Intuition>

<InterviewTip>
  Grid DP often uses a little table. Fill the easy edge cells first, then fill
  the inside cells from earlier neighbors.
</InterviewTip>

## TypeScript Solution

```ts
function uniquePaths(m: number, n: number): number {
  const dp = Array.from({ length: m }, () => new Array(n).fill(1));

  for (let row = 1; row < m; row += 1) {
    for (let col = 1; col < n; col += 1) {
      dp[row][col] = dp[row - 1][col] + dp[row][col - 1];
    }
  }

  return dp[m - 1][n - 1];
}
```
