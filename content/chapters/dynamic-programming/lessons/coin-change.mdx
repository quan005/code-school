<ProblemStatement title="Use the fewest coins to make the amount">
  Given coin values and a target amount, return the minimum number of coins
  needed to make that amount. Return <code>-1</code> if it is impossible.
</ProblemStatement>

## Example

```txt
coins = [1, 2, 5]
amount = 11
answer = 3
```

Because:

```txt
11 = 5 + 5 + 1
```

<Intuition title="Solve every smaller amount first">
  If we know the minimum coins for smaller amounts, then amount 11 can try each
  coin and ask, “What was the best answer for the smaller amount before this
  coin?”
</Intuition>

<InterviewTip>
  Coin Change is a minimum DP problem. The table cell for each amount stores the
  fewest coins known so far, not all possible combinations.
</InterviewTip>

## TypeScript Solution

```ts
function coinChange(coins: number[], amount: number): number {
  const dp = new Array(amount + 1).fill(Infinity);
  dp[0] = 0;

  for (let value = 1; value <= amount; value += 1) {
    for (const coin of coins) {
      if (value - coin >= 0) {
        dp[value] = Math.min(dp[value], dp[value - coin] + 1);
      }
    }
  }

  return dp[amount] === Infinity ? -1 : dp[amount];
}
```
