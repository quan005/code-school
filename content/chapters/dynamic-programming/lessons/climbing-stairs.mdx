<ProblemStatement title="How many different ways can you reach the top?">
  You are climbing a staircase with <code>n</code> steps. Each move can go up 1
  step or 2 steps. Return the number of different ways to reach the top.
</ProblemStatement>

## Example

```txt
n = 4
answer = 5
```

<Intuition title="The last move must come from one step before or two steps before">
  To land on step <code>i</code>, the final move must come from
  <code>i - 1</code> or <code>i - 2</code>. So the number of ways to reach step
  <code>i</code> is the sum of those two earlier counts.
</Intuition>

<InterviewTip>
  This problem feels like Fibonacci because each answer depends on the two steps
  before it.
</InterviewTip>

## TypeScript Solution

```ts
function climbStairs(n: number): number {
  if (n <= 2) {
    return n;
  }

  const dp = new Array(n + 1).fill(0);
  dp[1] = 1;
  dp[2] = 2;

  for (let step = 3; step <= n; step += 1) {
    dp[step] = dp[step - 1] + dp[step - 2];
  }

  return dp[n];
}
```
