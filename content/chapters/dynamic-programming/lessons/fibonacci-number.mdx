<ProblemStatement title="Return the nth Fibonacci number">
  Given <code>n</code>, return the <code>n</code>th Fibonacci number.
</ProblemStatement>

## Example

```txt
n = 6
answer = 8
```

Because the sequence is:

```txt
0, 1, 1, 2, 3, 5, 8
```

<Intuition title="Each answer uses the two before it">
  Once we know the first two base cases, each later Fibonacci number is just the
  sum of the previous two saved answers.
</Intuition>

<InterviewTip>
  Fibonacci is the friendliest first DP example because the rule is short and
  the repeated smaller work is easy to see.
</InterviewTip>

## TypeScript Solution

```ts
function fib(n: number): number {
  if (n === 0) {
    return 0;
  }

  if (n === 1) {
    return 1;
  }

  const dp = new Array(n + 1).fill(0);
  dp[0] = 0;
  dp[1] = 1;

  for (let index = 2; index <= n; index += 1) {
    dp[index] = dp[index - 1] + dp[index - 2];
  }

  return dp[n];
}
```
