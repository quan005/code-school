<ProblemStatement title="Reach the top as cheaply as possible">
  Given an array <code>cost</code>, where <code>cost[i]</code> is the price of
  stepping on step <code>i</code>, return the minimum cost to reach the top.
</ProblemStatement>

## Example

```txt
cost = [10, 15, 20]
answer = 15
```

<Intuition title="Now we want the cheaper path, not the number of paths">
  To reach a step, you still come from one step before or two steps before. The
  difference is that we keep the cheaper total cost instead of counting paths.
</Intuition>

<InterviewTip>
  Many DP problems reuse the same shape but change the question from “how many?”
  to “what is the minimum?” or “what is the maximum?”
</InterviewTip>

## TypeScript Solution

```ts
function minCostClimbingStairs(cost: number[]): number {
  const dp = new Array(cost.length).fill(0);
  dp[0] = cost[0];
  dp[1] = cost[1];

  for (let index = 2; index < cost.length; index += 1) {
    dp[index] = cost[index] + Math.min(dp[index - 1], dp[index - 2]);
  }

  return Math.min(dp[cost.length - 1], dp[cost.length - 2]);
}
```
