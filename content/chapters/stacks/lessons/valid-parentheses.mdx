<ProblemStatement title="Do the symbols close correctly?">
  Given a string with only <code>()[]{{}}</code> symbols, return{" "}
  <code>true</code> if every opening symbol is closed in the correct order.
  Otherwise, return <code>false</code>.
</ProblemStatement>

## Example

```txt
s = "([])"
answer = true
```

<Intuition title="The newest opening symbol must close first">
  When we see an opening symbol, we are waiting for its match. The most recent
  opening symbol is the one that must close next, which is exactly stack order.
</Intuition>

<InterviewTip>
  Counting symbols is not enough. The order matters too, which is why a stack
  fits better than a simple counter.
</InterviewTip>

## TypeScript Solution

```ts
function isValidParentheses(s: string): boolean {
  const stack: string[] = [];
  const pairs = new Map<string, string>([
    [")", "("],
    ["]", "["],
    ["}", "{"],
  ]);

  for (const ch of s) {
    if (ch === "(" || ch === "[" || ch === "{") {
      stack.push(ch);
    } else {
      if (stack.length === 0) {
        return false;
      }

      const top = stack.pop()!;
      if (top !== pairs.get(ch)) {
        return false;
      }
    }
  }

  return stack.length === 0;
}
```
