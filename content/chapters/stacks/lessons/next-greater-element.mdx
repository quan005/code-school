<ProblemStatement title="Find the next bigger value to the right">
  Given an array of numbers, return an array where each position tells the next
  greater number to the right. If there is none, use <code>-1</code>.
</ProblemStatement>

## Example

```txt
nums = [2, 1, 2, 4, 3]
answer = [4, 2, 4, -1, -1]
```

<Intuition title="Some positions are still waiting for an answer">
  We use a stack to store indexes whose next greater value is not known yet.
  When a bigger current value appears, it can solve the waiting positions on top
  of the stack.
</Intuition>

<InterviewTip>
  This is a more advanced stack pattern because the stack stores unfinished
  positions, not only direct values.
</InterviewTip>

## TypeScript Solution

```ts
function nextGreaterElement(nums: number[]): number[] {
  const answer = new Array(nums.length).fill(-1);
  const stack: number[] = [];

  for (let i = 0; i < nums.length; i++) {
    while (stack.length > 0 && nums[i] > nums[stack[stack.length - 1]]) {
      const index = stack.pop()!;
      answer[index] = nums[i];
    }

    stack.push(i);
  }

  return answer;
}
```
