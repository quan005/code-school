<ProblemStatement title="Erase matching neighbors">
  Given a string <code>s</code>, remove all adjacent matching letters again and
  again until no more such pairs exist. Return the final string.
</ProblemStatement>

## Example

```txt
s = "abbaca"
answer = "ca"
```

<Intuition title="The top stores the most recent survivor">
  If the current character matches the stack top, the two characters should
  disappear together. If not, the current character survives for now and gets
  pushed.
</Intuition>

<InterviewTip>
  This is a stack clue because every new character only needs to compare with
  the most recent surviving character.
</InterviewTip>

## TypeScript Solution

```ts
function removeAdjacentDuplicates(s: string): string {
  const stack: string[] = [];

  for (const ch of s) {
    if (stack.length > 0 && stack[stack.length - 1] === ch) {
      stack.pop();
    } else {
      stack.push(ch);
    }
  }

  return stack.join("");
}
```
