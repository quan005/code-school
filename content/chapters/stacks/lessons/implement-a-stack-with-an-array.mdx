<ProblemStatement title="Build the basic stack">
  Create a stack that supports <code>push</code>, <code>pop</code>,{" "}
  <code>peek</code>, and <code>isEmpty</code>.
</ProblemStatement>

## Example

```txt
start: []
push 3 -> [3]
push 8 -> [3, 8]
peek -> 8
pop -> 8
stack -> [3]
```

<Intuition title="Let the array end be the top">
  The easiest stack in TypeScript uses an array. The last spot in the array acts
  like the top of the stack.
</Intuition>

<InterviewTip>
  When you use an array as a stack, the right end is the top. That keeps the
  main actions simple.
</InterviewTip>

## TypeScript Solution

```ts
class MyStack {
  private items: number[] = [];

  push(x: number): void {
    this.items.push(x);
  }

  pop(): number | undefined {
    return this.items.pop();
  }

  peek(): number | undefined {
    if (this.items.length === 0) {
      return undefined;
    }

    return this.items[this.items.length - 1];
  }

  isEmpty(): boolean {
    return this.items.length === 0;
  }
}
```
