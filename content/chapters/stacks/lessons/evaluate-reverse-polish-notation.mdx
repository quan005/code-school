<ProblemStatement title="Solve the expression step by step">
  You are given an expression in Reverse Polish Notation. Numbers appear first,
  and operators come after them. Return the value of the expression.
</ProblemStatement>

## Example

```txt
tokens = ["2", "1", "+", "3", "*"]
answer = 9
```

<Intuition title="Numbers wait until an operator is ready">
  When we see a number, it goes onto the stack. When we see an operator, we pop
  the top two numbers, do the math, and push the result back.
</Intuition>

<InterviewTip>
  The stack holds unfinished numbers and partial answers. Each operator turns
  two top values into one new value.
</InterviewTip>

## TypeScript Solution

```ts
function evalRPN(tokens: string[]): number {
  const stack: number[] = [];

  for (const token of tokens) {
    if (token === "+" || token === "-" || token === "*" || token === "/") {
      const b = stack.pop()!;
      const a = stack.pop()!;
      let result = 0;

      if (token === "+") {
        result = a + b;
      } else if (token === "-") {
        result = a - b;
      } else if (token === "*") {
        result = a * b;
      } else {
        result = Math.trunc(a / b);
      }

      stack.push(result);
    } else {
      stack.push(Number(token));
    }
  }

  return stack[0];
}
```
