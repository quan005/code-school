<ProblemStatement title="Use the most recent scores">
  You are given a list of score operations. A number adds that score,{" "}
  <code>"C"</code> removes the last score, <code>"D"</code> doubles the last
  score, and <code>"+"</code> adds the last two scores. Return the total score.
</ProblemStatement>

## Example

```txt
ops = ["5", "2", "C", "D", "+"]
answer = 30
```

<Intuition title="The rules keep asking about recent history">
  This problem keeps asking for the newest score or the newest two scores. That
  makes a stack a strong fit because the top always holds the most recent score.
</Intuition>

<InterviewTip>
  If the rules keep saying last score or last two scores, think about using a
  stack to store history.
</InterviewTip>

## TypeScript Solution

```ts
function calPoints(ops: string[]): number {
  const stack: number[] = [];

  for (const op of ops) {
    if (op === "C") {
      stack.pop();
    } else if (op === "D") {
      stack.push(stack[stack.length - 1] * 2);
    } else if (op === "+") {
      const last = stack[stack.length - 1];
      const secondLast = stack[stack.length - 2];
      stack.push(last + secondLast);
    } else {
      stack.push(Number(op));
    }
  }

  let total = 0;

  for (const score of stack) {
    total += score;
  }

  return total;
}
```
