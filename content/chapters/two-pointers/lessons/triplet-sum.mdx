<ProblemStatement title="Find any matching triplet">
  Given an array of integers and a target value, return `true` if any three
  numbers add up to the target. Otherwise, return `false`.
</ProblemStatement>

## Examples

```txt
nums = [-1, 0, 2, 3], target = 2 -> true
nums = [1, 2, 4, 9], target = 20 -> false
```

<Intuition title="One loop plus Pair Sum">
  This sounds like a bigger problem, but it is really: pick one number, then
  find a pair that makes the rest. That means one loop plus the two-pointer
  trick.
</Intuition>

<InterviewTip>
  Sort first. After that, every choice for the first number turns into a smaller
  pair-sum search.
</InterviewTip>

## Brute Force Idea

The slow idea checks every possible triplet. That works, but it can take about
`O(n³)` time because there are so many combinations.

## Better Plan

1. Sort the array.
2. Use `i` to choose the first number.
3. Use `left` and `right` to find the other two numbers.

## Walkthrough

For `nums = [-1, 0, 2, 3]` and `target = 2`:

1. Pick `i = 0`, so the first number is `-1`.
2. Now we need two numbers that add up to `3`.
3. Set `left = 1` and `right = 3`.
4. `nums[left] + nums[right] = 0 + 3 = 3`, so the triplet exists.

## TypeScript Solution

```ts
function tripletSum(nums: number[], target: number): boolean {
  nums = [...nums].sort((a, b) => a - b);

  for (let i = 0; i < nums.length - 2; i++) {
    let left = i + 1;
    let right = nums.length - 1;
    const needed = target - nums[i];

    while (left < right) {
      const sum = nums[left] + nums[right];

      if (sum < needed) {
        left += 1;
      } else if (sum > needed) {
        right -= 1;
      } else {
        return true;
      }
    }
  }

  return false;
}
```
