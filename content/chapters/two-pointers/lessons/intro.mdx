<ProblemStatement title="Two pointers in one sentence">
  Two pointers means keeping track of two places in the same list at the same
  time.
</ProblemStatement>

<Intuition title="Think of two fingers on a number line">
  Put one finger on the left and one on the right. Each move should teach you
  something. In a sorted array, moving one side can throw away many impossible
  answers without checking them one by one.
</Intuition>

<InterviewTip>
  When the list is sorted, ask: "If my sum is too small, which side can make it
  bigger?" That question usually tells you which pointer should move.
</InterviewTip>

<ComplexityCard
  time="O(n)"
  space="O(1)"
  notes="Each pointer moves across the list at most once, so the work grows in a straight line."
/>

## The Three Main Styles

- **Inward traversal** starts at both ends and moves toward the middle.
- **Unidirectional traversal** moves two pointers the same direction.
- **Staged traversal** finds one important spot first, then finishes the job.

### Real-world picture

Imagine cleaning up a shelf. One finger scans every box. Another finger marks
the next good spot where a useful box should go. That is a real two-pointer
idea.

## Tiny Practice

You have a sorted array `nums = [1, 3, 5, 8, 10]`.

If `left = 0` and `right = 4`, then the values are `1` and `10`. If the target
sum is `13`, the total is `11`, which is too small.

**What should move next?** Move `left` to the right so the sum can get bigger.

## Lesson Summary

- A pointer is a variable that stores a position.
- Two pointers help us compare and move with a reason.
- The three main styles are inward, unidirectional, and staged traversal.
- The best clue is a pattern you can trust, like sorted order or symmetry.
