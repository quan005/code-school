<ProblemStatement title="Check whether a word reads the same both ways">
  Return `true` if a string is a palindrome and `false` otherwise.
</ProblemStatement>

## Examples

```txt
"racecar" -> true
"rocket" -> false
```

<Intuition title="Symmetry is the clue">
  A palindrome is a great fit for two pointers because the first and last
  letters should match, then the next two, and so on.
</Intuition>

<InterviewTip>
  You do not need to build a reversed copy to solve this. Two pointers can
  compare the word in place.
</InterviewTip>

## Plan

1. Put `left` at the start.
2. Put `right` at the end.
3. Compare the letters.
4. If they do not match, return `false`.
5. If they do match, move inward.

## Walkthrough

For `"racecar"`:

1. Compare `r` and `r`.
2. Compare `a` and `a`.
3. Compare `c` and `c`.
4. Every pair matches, so the word is a palindrome.

## TypeScript Solution

```ts
function isPalindromeValid(word: string): boolean {
  let left = 0;
  let right = word.length - 1;

  while (left < right) {
    if (word[left] !== word[right]) {
      return false;
    }

    left += 1;
    right -= 1;
  }

  return true;
}
```

<HintDrawer title="What makes this an inward traversal problem?">
  The useful information lives at both ends of the word, so we compare from the
  outside and move toward the middle.
</HintDrawer>
