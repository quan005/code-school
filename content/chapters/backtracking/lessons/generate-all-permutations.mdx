<ProblemStatement title="Return every ordering">
  Given an array of distinct numbers <code>nums</code>, return all possible
  permutations.
</ProblemStatement>

## Example

```txt
nums = [1, 2, 3]
answer includes [1, 2, 3], [1, 3, 2], [2, 1, 3], and more
```

<Intuition title="This time the question is who goes next">
  A subset problem asks take or skip. A permutation problem asks which unused
  number should go in the next open spot.
</Intuition>

<InterviewTip>
  A <code>used</code> array or set is the clue that tells which choices are
  already inside the current path.
</InterviewTip>

## TypeScript Solution

```ts
function permute(nums: number[]): number[][] {
  const results: number[][] = [];
  const used = new Array(nums.length).fill(false);

  function backtrack(path: number[]): void {
    if (path.length === nums.length) {
      results.push([...path]);
      return;
    }

    for (let index = 0; index < nums.length; index += 1) {
      if (used[index]) {
        continue;
      }

      used[index] = true;
      path.push(nums[index]);

      backtrack(path);

      path.pop();
      used[index] = false;
    }
  }

  backtrack([]);
  return results;
}
```
