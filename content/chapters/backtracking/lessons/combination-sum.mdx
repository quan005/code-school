<ProblemStatement title="Return all combinations that reach the target">
  Given distinct numbers <code>candidates</code> and a target number, return all
  unique combinations where the chosen numbers add up to the target. You may use
  the same number more than once.
</ProblemStatement>

## Example

```txt
candidates = [2, 3, 6, 7]
target = 7
answer = [[2, 2, 3], [7]]
```

<Intuition title="Choose a number, then see how much target is left">
  When we pick a number, we subtract it from the remaining target. If the
  remaining target becomes zero, we found a good answer. If it becomes negative,
  that branch cannot work.
</Intuition>

<InterviewTip>
  In this problem, staying on the same index means the same number can be reused
  again.
</InterviewTip>

## TypeScript Solution

```ts
function combinationSum(candidates: number[], target: number): number[][] {
  const results: number[][] = [];

  function backtrack(index: number, remaining: number, path: number[]): void {
    if (remaining === 0) {
      results.push([...path]);
      return;
    }

    if (remaining < 0 || index === candidates.length) {
      return;
    }

    path.push(candidates[index]);
    backtrack(index, remaining - candidates[index], path);
    path.pop();

    backtrack(index + 1, remaining, path);
  }

  backtrack(0, target, []);
  return results;
}
```
