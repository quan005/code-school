<ProblemStatement title="Return every possible letter combination">
  Given a string of digits from <code>2</code> to <code>9</code>, return all
  possible letter combinations that the number could represent.
</ProblemStatement>

## Example

```txt
digits = "23"
answer = ["ad", "ae", "af", "bd", "be", "bf", "cd", "ce", "cf"]
```

<Intuition title="Each digit offers a small menu of letters">
  One digit gives a few possible letters. We choose one letter for the current
  spot, explore deeper, then come back and try the next letter.
</Intuition>

<InterviewTip>
  This is a strong backtracking clue because the answer must return all possible
  strings built from repeated small choices.
</InterviewTip>

## TypeScript Solution

```ts
function letterCombinations(digits: string): string[] {
  if (digits.length === 0) {
    return [];
  }

  const map = new Map<string, string>([
    ["2", "abc"],
    ["3", "def"],
    ["4", "ghi"],
    ["5", "jkl"],
    ["6", "mno"],
    ["7", "pqrs"],
    ["8", "tuv"],
    ["9", "wxyz"],
  ]);

  const results: string[] = [];

  function backtrack(index: number, path: string): void {
    if (index === digits.length) {
      results.push(path);
      return;
    }

    const letters = map.get(digits[index]) ?? "";

    for (const letter of letters) {
      backtrack(index + 1, path + letter);
    }
  }

  backtrack(0, "");
  return results;
}
```
