<ProblemStatement title="Return every valid parentheses string">
  Given <code>n</code> pairs of parentheses, return all valid combinations.
</ProblemStatement>

## Example

```txt
n = 3
answer includes ((())), (()()), (())(), ()(()), ()()()
```

<Intuition title="Some partial paths are already broken">
  We can only place a closing parenthesis if there is already an unmatched open
  one to close. That lets us stop bad branches early instead of exploring
  nonsense paths.
</Intuition>

<InterviewTip>
  A branch that is already impossible should stop right away. That early stop is
  often called pruning.
</InterviewTip>

## TypeScript Solution

```ts
function generateParenthesis(n: number): string[] {
  const results: string[] = [];

  function backtrack(path: string, open: number, close: number): void {
    if (path.length === n * 2) {
      results.push(path);
      return;
    }

    if (open < n) {
      backtrack(path + "(", open + 1, close);
    }

    if (close < open) {
      backtrack(path + ")", open, close + 1);
    }
  }

  backtrack("", 0, 0);
  return results;
}
```
