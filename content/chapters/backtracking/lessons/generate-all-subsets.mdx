<ProblemStatement title="Return every subset">
  Given an array of distinct numbers <code>nums</code>, return all possible
  subsets.
</ProblemStatement>

## Example

```txt
nums = [1, 2]
answer = [[], [1], [2], [1, 2]]
```

<Intuition title="Each number asks one yes-or-no question">
  For every number, we have two choices: take it or skip it. Those two choices
  make two branches in the decision tree.
</Intuition>

<InterviewTip>
  Subset problems are one of the clearest backtracking patterns because each
  item creates a simple include-or-skip choice.
</InterviewTip>

## TypeScript Solution

```ts
function subsets(nums: number[]): number[][] {
  const results: number[][] = [];

  function backtrack(index: number, path: number[]): void {
    if (index === nums.length) {
      results.push([...path]);
      return;
    }

    path.push(nums[index]);
    backtrack(index + 1, path);
    path.pop();

    backtrack(index + 1, path);
  }

  backtrack(0, []);
  return results;
}
```
