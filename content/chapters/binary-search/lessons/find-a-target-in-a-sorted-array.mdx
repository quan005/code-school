<ProblemStatement title="Find the exact spot">
  Given a sorted array of integers <code>nums</code> and a target value, return
  the index of the target if it exists. Otherwise, return <code>-1</code>.
</ProblemStatement>

## Example

```txt
nums = [1, 3, 5, 7, 9], target = 7
answer = 3
```

<Intuition title="The middle tells us which side still matters">
  Because the array is sorted, the middle number gives a clue. If the middle is
  too small, we keep the right half. If the middle is too big, we keep the left
  half.
</Intuition>

<InterviewTip>
  The key promise of sorted order is this: after one comparison, one whole half
  becomes impossible.
</InterviewTip>

## TypeScript Solution

```ts
function binarySearch(nums: number[], target: number): number {
  let left = 0;
  let right = nums.length - 1;

  while (left <= right) {
    const mid = Math.floor((left + right) / 2);

    if (nums[mid] === target) {
      return mid;
    } else if (nums[mid] < target) {
      left = mid + 1;
    } else {
      right = mid - 1;
    }
  }

  return -1;
}
```
