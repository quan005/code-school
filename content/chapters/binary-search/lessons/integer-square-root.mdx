<ProblemStatement title="Find the whole-number square root">
  Given a non-negative integer <code>x</code>, return the integer part of its
  square root. Do not use a built-in square root function.
</ProblemStatement>

## Example

```txt
x = 8
answer = 2
```

<Intuition title="We are searching for the biggest square that fits">
  We are not searching inside an array this time. We are searching the answer
  range from <code>0</code> to <code>x</code>. The middle guess tells us whether
  its square is too small, too big, or perfect.
</Intuition>

<InterviewTip>
  Binary search can work on any ordered range where you can test whether a guess
  is too small or too large.
</InterviewTip>

## TypeScript Solution

```ts
function integerSqrt(x: number): number {
  let left = 0;
  let right = x;
  let answer = 0;

  while (left <= right) {
    const mid = Math.floor((left + right) / 2);
    const square = mid * mid;

    if (square === x) {
      return mid;
    } else if (square < x) {
      answer = mid;
      left = mid + 1;
    } else {
      right = mid - 1;
    }
  }

  return answer;
}
```

  the answer range to keep.
</HintDrawer>
