<ProblemStatement title="Play the smart guessing game">
  A secret number is chosen from <code>1</code> to <code>n</code>. You can ask
  whether your guess is correct, too high, or too low. Return the secret number
  using as few guesses as possible.
</ProblemStatement>

## Example

```txt
n = 10, secret = 6
answer = 6
```

<Intuition title="Every clue cuts away half">
  This is binary search in game form. Guess the middle. If the clue says too
  low, keep the higher half. If the clue says too high, keep the lower half.
</Intuition>

<InterviewTip>
  When a problem lets you ask “too high or too low,” it often means binary
  search is the right idea.
</InterviewTip>

## TypeScript Solution

```ts
declare function guess(num: number): -1 | 0 | 1;

function guessNumber(n: number): number {
  let left = 1;
  let right = n;

  while (left <= right) {
    const mid = Math.floor((left + right) / 2);
    const result = guess(mid);

    if (result === 0) {
      return mid;
    } else if (result === 1) {
      left = mid + 1;
    } else {
      right = mid - 1;
    }
  }

  return -1;
}
```
