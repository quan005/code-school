<ProblemStatement title="Multiply everything except the current spot">
  Given an array <code>nums</code>, return an array where each answer at index{" "}
  <code>i</code> is the product of all numbers except <code>nums[i]</code>. Do
  not use division.
</ProblemStatement>

## Example

```txt
nums = [1, 2, 3, 4]
answer = [24, 12, 8, 6]
```

<Intuition title="Think left side and right side">
  This is like prefix sums, but with multiplication. For each position, we want:
</Intuition>

```txt
product before it * product after it
```

<InterviewTip>
  This lesson belongs near prefix sums because it uses the same left-side and
  right-side precomputing idea, just with products instead of sums.
</InterviewTip>

## TypeScript Solution

```ts
function productExceptSelf(nums: number[]): number[] {
  const result = new Array(nums.length).fill(1);

  let prefix = 1;
  for (let i = 0; i < nums.length; i++) {
    result[i] = prefix;
    prefix *= nums[i];
  }

  let suffix = 1;
  for (let i = nums.length - 1; i >= 0; i--) {
    result[i] *= suffix;
    suffix *= nums[i];
  }

  return result;
}
```

  multiply in what is on the right.
