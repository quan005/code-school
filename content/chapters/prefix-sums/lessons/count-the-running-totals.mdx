<ProblemStatement title="Store the total so far at each position">
  Given an array of numbers, return an array where each position stores the sum
  from the beginning up to that point.
</ProblemStatement>

## Example

```txt
nums = [1, 2, 3, 4]
answer = [1, 3, 6, 10]
```

<Intuition title="Add one new number to the total each step">
  This is the simplest prefix-sum problem. We keep one running total and save it
  after every new number.
</Intuition>

<InterviewTip>
  A running total is prefix-sum thinking in its simplest form: what is the total
  so far?
</InterviewTip>

## TypeScript Solution

```ts
function runningTotals(nums: number[]): number[] {
  const result: number[] = [];
  let total = 0;

  for (const num of nums) {
    total += num;
    result.push(total);
  }

  return result;
}
```
