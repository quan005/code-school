<ProblemStatement title="Find the balance point">
  Given an array <code>nums</code>, return the pivot index where the sum on the
  left equals the sum on the right. If none exists, return <code>-1</code>.
</ProblemStatement>

## Example

```txt
nums = [1, 7, 3, 6, 5, 6]
answer = 3
```

<Intuition title="Use one whole total and one running left total">
  We do not need to rebuild the left and right sums at every position. The total
  sum tells us the whole array, and the running left sum tells us the left side.
  The right side becomes what is left over.
</Intuition>

<InterviewTip>
  This is a common pattern: total sum plus running left sum equals quick access
  to the right sum.
</InterviewTip>

## TypeScript Solution

```ts
function pivotIndex(nums: number[]): number {
  let total = 0;

  for (const num of nums) {
    total += num;
  }

  let leftSum = 0;

  for (let i = 0; i < nums.length; i++) {
    const rightSum = total - leftSum - nums[i];

    if (leftSum === rightSum) {
      return i;
    }

    leftSum += nums[i];
  }

  return -1;
}
```
