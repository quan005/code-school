<ProblemStatement title="Answer many sum questions quickly">
  Given an array <code>nums</code>, answer queries asking for the sum from index{" "}
  <code>left</code> to index <code>right</code>.
</ProblemStatement>

## Example

```txt
nums = [2, 4, 1, 3, 5]
left = 1
right = 3
answer = 8
```

<Intuition title="Use a helper array instead of re-adding every time">
  If many queries are coming, it is better to build a prefix sum array once.
  Then each range sum can be answered with one subtraction.
</Intuition>

<InterviewTip>
  Prefix sums shine when many questions reuse the same array and the brute-force
  method would keep adding the same values again and again.
</InterviewTip>

## TypeScript Solution

```ts
function buildPrefix(nums: number[]): number[] {
  if (nums.length === 0) {
    return [];
  }

  const prefix = new Array(nums.length);
  prefix[0] = nums[0];

  for (let i = 1; i < nums.length; i++) {
    prefix[i] = prefix[i - 1] + nums[i];
  }

  return prefix;
}

function rangeSum(prefix: number[], left: number, right: number): number {
  if (left === 0) {
    return prefix[right];
  }

  return prefix[right] - prefix[left - 1];
}
```
