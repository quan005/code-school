<ProblemStatement title="How many side-by-side groups make the goal sum?">
  Given an array <code>nums</code> and a goal sum <code>k</code>, return how
  many subarrays have a sum equal to <code>k</code>.
</ProblemStatement>

## Example

```txt
nums = [1, 1, 1]
k = 2
answer = 2
```

<Intuition title="Look for an earlier total that makes the difference work">
  Let <code>currentSum</code> be the running total so far. If a subarray should
  add to <code>k</code>, then we want an earlier prefix sum so that:
</Intuition>

```txt
currentSum - earlierPrefix = k
```

<InterviewTip>
  This is one of the most important “prefix sum + hash map” patterns. The map
  stores how many times each earlier prefix sum has appeared.
</InterviewTip>

## TypeScript Solution

```ts
function subarraySumEqualsK(nums: number[], k: number): number {
  const counts = new Map<number, number>();
  counts.set(0, 1);

  let currentSum = 0;
  let totalCount = 0;

  for (const num of nums) {
    currentSum += num;

    const need = currentSum - k;
    totalCount += counts.get(need) ?? 0;

    counts.set(currentSum, (counts.get(currentSum) ?? 0) + 1);
  }

  return totalCount;
}
```
