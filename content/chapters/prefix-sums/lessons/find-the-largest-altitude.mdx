<ProblemStatement title="Track the highest point reached">
  A biker starts at altitude 0. You are given gains and losses after each step.
  Return the highest altitude the biker reaches.
</ProblemStatement>

## Example

```txt
gain = [-5, 1, 5, 0, -7]
answer = 1
```

<Intuition title="Altitude is just a running total">
  Each new altitude is the old altitude plus the next gain or loss. That means
  altitude behaves like a prefix sum of the gain array.
</Intuition>

<InterviewTip>
  Not every prefix-sum problem needs a full prefix array. Sometimes one running
  total and one best-so-far value are enough.
</InterviewTip>

## TypeScript Solution

```ts
function largestAltitude(gain: number[]): number {
  let altitude = 0;
  let highest = 0;

  for (const change of gain) {
    altitude += change;
    highest = Math.max(highest, altitude);
  }

  return highest;
}
```
