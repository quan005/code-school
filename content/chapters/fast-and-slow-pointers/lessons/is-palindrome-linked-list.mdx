<ProblemStatement title="Check whether the list reads the same both ways">
  Given the <code>head</code> of a linked list, return <code>true</code> if the
  list reads the same forward and backward. Otherwise, return <code>false</code>
  .
</ProblemStatement>

## Examples

```txt
1 -> 2 -> 2 -> 1 -> null -> true
1 -> 2 -> 3 -> null -> false
```

<Intuition title="Find the middle, then compare halves">
  Linked lists cannot easily look backward, so we use fast and slow pointers to
  find the middle, reverse the second half, and then compare the two halves.
</Intuition>

<InterviewTip>
  This lesson combines two patterns: fast-and-slow pointers and linked-list
  reversal.
</InterviewTip>

## TypeScript Solution

```ts
function reverseList(head: ListNode | null): ListNode | null {
  let prev: ListNode | null = null;
  let current = head;

  while (current !== null) {
    const nextNode = current.next;
    current.next = prev;
    prev = current;
    current = nextNode;
  }

  return prev;
}

function isPalindromeList(head: ListNode | null): boolean {
  if (head === null || head.next === null) {
    return true;
  }

  let slow: ListNode | null = head;
  let fast: ListNode | null = head;

  while (fast !== null && fast.next !== null) {
    slow = slow!.next;
    fast = fast.next.next;
  }

  let secondHalf = reverseList(slow);
  let firstHalf = head;

  while (secondHalf !== null) {
    if (firstHalf!.val !== secondHalf.val) {
      return false;
    }

    firstHalf = firstHalf!.next;
    secondHalf = secondHalf.next;
  }

  return true;
}
```
