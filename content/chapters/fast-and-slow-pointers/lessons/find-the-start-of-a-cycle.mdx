<ProblemStatement title="Return the node where the cycle begins">
  Given the <code>head</code> of a linked list, return the node where the cycle
  begins. If there is no cycle, return <code>null</code>.
</ProblemStatement>

## Example

```txt
1 -> 2 -> 3 -> 4 -> 5
          ^         |
          |_________|
cycle starts at 3
```

<Intuition title="Meeting first, then walk together">
  First use fast and slow pointers to find a meeting point inside the loop. Then
  put one pointer at the head and keep the other at the meeting point. Move both
  1 step at a time. Where they meet is the cycle start.
</Intuition>

<InterviewTip>
  This is the chapter’s trickiest lesson. The pattern matters more than the math
  story at first: meet inside the loop, then walk together.
</InterviewTip>

## TypeScript Solution

```ts
function detectCycleStart(head: ListNode | null): ListNode | null {
  let slow = head;
  let fast = head;

  while (fast !== null && fast.next !== null) {
    slow = slow!.next;
    fast = fast.next.next;

    if (slow === fast) {
      let pointer1 = head;
      let pointer2 = slow;

      while (pointer1 !== pointer2) {
        pointer1 = pointer1!.next;
        pointer2 = pointer2!.next;
      }

      return pointer1;
    }
  }

  return null;
}
```
