<ProblemStatement title="Decide whether the number reaches 1">
  A number is <strong>happy</strong> if repeatedly squaring its digits and
  adding them eventually reaches 1. Return <code>true</code> if <code>n</code>{" "}
  is happy, otherwise return <code>false</code>.
</ProblemStatement>

## Example

```txt
19 -> 82 -> 68 -> 100 -> 1
```

<Intuition title="This is a path, even without a linked list">
  Each number turns into the next number, and then the next one after that. That
  makes a path of states. The path can either reach 1 or get stuck in a loop,
  just like a linked list can end or cycle.
</Intuition>

<InterviewTip>
  Fast and slow pointers are not only for linked lists. They also work on any
  repeated process that creates one next state from the current one.
</InterviewTip>

## Helper Function

```ts
function nextNumber(n: number): number {
  let total = 0;

  while (n > 0) {
    const digit = n % 10;
    total += digit * digit;
    n = Math.floor(n / 10);
  }

  return total;
}
```

## TypeScript Solution

```ts
function nextNumber(n: number): number {
  let total = 0;

  while (n > 0) {
    const digit = n % 10;
    total += digit * digit;
    n = Math.floor(n / 10);
  }

  return total;
}

function isHappy(n: number): boolean {
  let slow = n;
  let fast = n;

  while (true) {
    slow = nextNumber(slow);
    fast = nextNumber(nextNumber(fast));

    if (fast === 1) {
      return true;
    }

    if (slow === fast) {
      return false;
    }
  }
}
```
