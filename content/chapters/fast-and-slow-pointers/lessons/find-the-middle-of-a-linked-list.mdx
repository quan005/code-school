<ProblemStatement title="Return the middle node">
  Given the <code>head</code> of a linked list, return the middle node. If there
  are two middle nodes, return the second one.
</ProblemStatement>

## Examples

```txt
1 -> 2 -> 3 -> null -> middle is 2
1 -> 2 -> 3 -> 4 -> null -> middle is 3
```

<Intuition title="Fast finishes while slow reaches halfway">
  If fast takes 2 steps every turn and slow takes 1, then when fast reaches the
  end, slow has only traveled about half as far.
</Intuition>

<InterviewTip>
  This beats the two-pass plan of counting everything first and then walking
  again.
</InterviewTip>

## TypeScript Solution

```ts
function middleNode(head: ListNode | null): ListNode | null {
  let slow = head;
  let fast = head;

  while (fast !== null && fast.next !== null) {
    slow = slow!.next;
    fast = fast.next.next;
  }

  return slow;
}
```
