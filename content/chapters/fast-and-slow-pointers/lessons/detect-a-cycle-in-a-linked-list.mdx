<ProblemStatement title="Check whether the list loops back">
  Given the <code>head</code> of a linked list, return <code>true</code> if the
  list contains a cycle. Otherwise, return <code>false</code>.
</ProblemStatement>

## Example

```txt
1 -> 2 -> 3 -> 4
     ^         |
     |_________|
```

<Intuition title="Fast catches slow on a loop">
  If there is no cycle, fast eventually falls off the end at null. If there is a
  cycle, both pointers run around the same loop, and the faster one eventually
  catches the slower one.
</Intuition>

<InterviewTip>
  You do not need a set for this pattern. The meeting itself is the clue.
</InterviewTip>

## TypeScript Solution

```ts
function hasCycle(head: ListNode | null): boolean {
  let slow = head;
  let fast = head;

  while (fast !== null && fast.next !== null) {
    slow = slow!.next;
    fast = fast.next.next;

    if (slow === fast) {
      return true;
    }
  }

  return false;
}
```
