<ProblemStatement title="Sort the array with selection sort">
  Given an array of numbers, sort it in ascending order using selection sort.
</ProblemStatement>

## Example

```txt
[5, 3, 4, 1, 2] -> [1, 2, 3, 4, 5]
```

<Intuition title="Put the next smallest value into the next correct spot">
  Selection sort looks through the unsorted part, finds the smallest value, and
  swaps it into the next open sorted position on the left.
</Intuition>

<InterviewTip>
  Selection sort is easy to understand because it tells a very clear story: find
  the next smallest, place it, then repeat.
</InterviewTip>

## TypeScript Solution

```ts
function selectionSort(nums: number[]): number[] {
  const arr = [...nums];

  for (let left = 0; left < arr.length; left += 1) {
    let minIndex = left;

    for (let scan = left + 1; scan < arr.length; scan += 1) {
      if (arr[scan] < arr[minIndex]) {
        minIndex = scan;
      }
    }

    [arr[left], arr[minIndex]] = [arr[minIndex], arr[left]];
  }

  return arr;
}
```
