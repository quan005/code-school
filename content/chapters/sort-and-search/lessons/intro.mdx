<ProblemStatement title="Searching finds things and sorting puts things in order">
  Searching means looking for something. Sorting means arranging things in a
  clear order.
</ProblemStatement>

<Intuition title="Think about a messy box of cards">
  If the cards are messy, you can still search one by one. But if the cards are
  sorted, it becomes easier to compare neighbors, merge lists, and place new
  cards in the right spot.
</Intuition>

<InterviewTip>
  Searching asks things like “Is this value here?” Sorting asks things like “Can
  we put these values in a helpful order first?”
</InterviewTip>

## Helpful ideas

- <strong>searching</strong>: look for a value or check whether it exists
- <strong>sorting</strong>: arrange items in a clear order
- <strong>order</strong>: a pattern like smallest to largest
- <strong>structure</strong>: the helpful shape sorted data gives us

## Strong sort-and-search clues

- find a value
- put values in order
- compare neighbors
- merge sorted lists
- place something where it belongs

## Lesson Summary

- Searching looks for something.
- Sorting arranges things in order.
- Sorted order can make later work easier.
- Sometimes direct search is enough.
- Sometimes sorting first is the smarter plan.
