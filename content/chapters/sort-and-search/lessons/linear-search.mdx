<ProblemStatement title="Find the target by checking one item at a time">
  Given an array <code>nums</code> and a target value, return the target’s index
  if it exists. Otherwise, return <code>-1</code>.
</ProblemStatement>

## Example

```txt
nums = [4, 2, 7, 1]
target = 7
answer = 2
```

<Intuition title="Start at the beginning and scan across">
  Linear search does not need sorted order. It just checks each position one by
  one until it finds the target or runs out of items.
</Intuition>

<InterviewTip>
  Linear search is a good beginner tool when the data is small or unsorted and
  you simply need to check item by item.
</InterviewTip>

## TypeScript Solution

```ts
function linearSearch(nums: number[], target: number): number {
  for (let index = 0; index < nums.length; index += 1) {
    if (nums[index] === target) {
      return index;
    }
  }

  return -1;
}
```
