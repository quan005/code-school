<ProblemStatement title="Combine two sorted arrays into one sorted array">
  Given two arrays already sorted in ascending order, merge them into one sorted
  array.
</ProblemStatement>

## Example

```txt
a = [1, 3, 5]
b = [2, 4, 6]
answer = [1, 2, 3, 4, 5, 6]
```

<Intuition title="Sorted order means the next smallest value is right at the front">
  Because both arrays are already sorted, the next answer must be whichever
  current front value is smaller.
</Intuition>

<InterviewTip>
  This is a great example of how sorting first can make a later task much
  easier.
</InterviewTip>

## TypeScript Solution

```ts
function mergeSortedArrays(a: number[], b: number[]): number[] {
  const result: number[] = [];
  let left = 0;
  let right = 0;

  while (left < a.length && right < b.length) {
    if (a[left] <= b[right]) {
      result.push(a[left]);
      left += 1;
    } else {
      result.push(b[right]);
      right += 1;
    }
  }

  while (left < a.length) {
    result.push(a[left]);
    left += 1;
  }

  while (right < b.length) {
    result.push(b[right]);
    right += 1;
  }

  return result;
}
```
