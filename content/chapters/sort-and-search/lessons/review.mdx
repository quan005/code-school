## Pattern Summary

- Linear Search: check one by one.
- Selection Sort: place the next smallest value.
- Insertion Sort: slide one value into the sorted side.
- Merge Two Sorted Arrays: compare the two current fronts.
- Sort Colors: count and rewrite because there are only three values.
- Rotated Search: use the ordered half that still remains.
