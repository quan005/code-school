<ProblemStatement title="Sort an array containing only 0, 1, and 2">
  Given an array containing only <code>0</code>, <code>1</code>, and
  <code>2</code>, sort it in place so all 0s come first, then 1s, then 2s.
</ProblemStatement>

## Example

```txt
[2, 0, 2, 1, 1, 0] -> [0, 0, 1, 1, 2, 2]
```

<Intuition title="Counting is enough because there are only three kinds">
  This problem is easier than general sorting because the array contains only
  three possible values. Count how many of each value there are, then write them
  back in order.
</Intuition>

<InterviewTip>
  When the set of possible values is tiny, counting can sometimes be easier than
  using a more general sorting algorithm.
</InterviewTip>

## TypeScript Solution

```ts
function sortColors(nums: number[]): void {
  let zeros = 0;
  let ones = 0;
  let twos = 0;

  for (const num of nums) {
    if (num === 0) {
      zeros += 1;
    } else if (num === 1) {
      ones += 1;
    } else {
      twos += 1;
    }
  }

  let index = 0;

  for (let count = 0; count < zeros; count += 1) {
    nums[index] = 0;
    index += 1;
  }

  for (let count = 0; count < ones; count += 1) {
    nums[index] = 1;
    index += 1;
  }

  for (let count = 0; count < twos; count += 1) {
    nums[index] = 2;
    index += 1;
  }
}
```
