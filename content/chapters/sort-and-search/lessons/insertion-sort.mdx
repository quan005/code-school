<ProblemStatement title="Sort the array with insertion sort">
  Given an array of numbers, sort it in ascending order using insertion sort.
</ProblemStatement>

## Example

```txt
[5, 3, 4, 1, 2] -> [1, 2, 3, 4, 5]
```

<Intuition title="Sort like cards in your hand">
  Insertion sort treats the left side as already sorted. Then it takes one new
  value and slides it left until it fits in the correct spot.
</Intuition>

<InterviewTip>
  If selection sort is “pick the next smallest,” insertion sort is “take one
  card and slide it into place.”
</InterviewTip>

## TypeScript Solution

```ts
function insertionSort(nums: number[]): number[] {
  const arr = [...nums];

  for (let index = 1; index < arr.length; index += 1) {
    const current = arr[index];
    let scan = index - 1;

    while (scan >= 0 && arr[scan] > current) {
      arr[scan + 1] = arr[scan];
      scan -= 1;
    }

    arr[scan + 1] = current;
  }

  return arr;
}
```
