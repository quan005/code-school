<ProblemStatement title="Check whether two words use the same letters">
  Two strings are anagrams if they use the same letters the same number of
  times. Return <code>true</code> if <code>t</code> is an anagram of{" "}
  <code>s</code>. Otherwise, return <code>false</code>.
</ProblemStatement>

## Examples

```txt
s = "listen", t = "silent" -> true
s = "rat", t = "car" -> false
```

<Intuition title="This is a counting problem">
  A set is not enough because a set would only tell us whether a letter exists.
  Anagrams need the exact number of each letter, so we use a map of letter to
  count.
</Intuition>

<InterviewTip>
  If two words have different lengths, you can return false right away. They
  cannot possibly use the same letters the same number of times.
</InterviewTip>

## Better Plan

1. If the lengths are different, return <code>false</code>.
2. Count the letters in the first word.
3. Walk through the second word and subtract from those counts.
4. If a letter is missing or goes below zero, return <code>false</code>.

## Walkthrough

For <code>s = "listen"</code> and <code>t = "silent"</code>:

1. Count the letters in <code>listen</code>.
2. Read <code>silent</code> one letter at a time.
3. Each matching letter lowers the correct count.
4. Because every letter balances correctly, the words are anagrams.

## TypeScript Solution

```ts
function validAnagram(s: string, t: string): boolean {
  if (s.length !== t.length) {
    return false;
  }

  const counts = new Map<string, number>();

  for (const ch of s) {
    counts.set(ch, (counts.get(ch) ?? 0) + 1);
  }

  for (const ch of t) {
    if (!counts.has(ch)) {
      return false;
    }

    counts.set(ch, counts.get(ch)! - 1);

    if (counts.get(ch)! < 0) {
      return false;
    }
  }

  return true;
}
```
