<ProblemStatement title="Find out whether any number repeats">
  Given an array of integers, return <code>true</code> if any value appears more
  than once. Return <code>false</code> if all values are different.
</ProblemStatement>

## Examples

```txt
nums = [3, 1, 4, 2, 1] -> true
nums = [5, 6, 7] -> false
```

<Intuition title="This is a seen-before problem">
  We do not need counts or indexes. We only need to know whether the current
  number has already shown up. That is exactly what a set is good at.
</Intuition>

<InterviewTip>
  When your first idea is “compare every number with every other number,” pause
  and ask if a set can remember what you already saw.
</InterviewTip>

## Brute Force Idea

The slow plan checks every pair of positions. That works, but it takes about

<code>O(n²)</code> time because there are so many comparisons.

## Better Plan

1. Start with an empty set called <code>seen</code>.
2. Read the numbers one by one.
3. If the number is already in <code>seen</code>, return <code>true</code>.
4. Otherwise, add the number to the set.

## Walkthrough

For <code>nums = [3, 1, 4, 2, 1]</code>:

1. Add <code>3</code> to the set.
2. Add <code>1</code> to the set.
3. Add <code>4</code> to the set.
4. Add <code>2</code> to the set.
5. See <code>1</code> again, notice it is already in the set, and stop.

## TypeScript Solution

```ts
function containsDuplicate(nums: number[]): boolean {
  const seen = new Set<number>();

  for (const num of nums) {
    if (seen.has(num)) {
      return true;
    }

    seen.add(num);
  }

  return false;
}
```
