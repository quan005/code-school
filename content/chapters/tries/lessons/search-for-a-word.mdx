<ProblemStatement title="Find out whether a whole word exists">
  Given a trie and a word, return <code>true</code> if the trie stores that
  whole word. Return <code>false</code> otherwise.
</ProblemStatement>

## Example

```txt
stored words: to, top, toy
search("to") -> true
search("tom") -> false
```

<Intuition title="Follow every letter, then check the finish line">
  Start at the root and follow one child per letter. If a letter path is
  missing, the word is not there. If the path exists all the way through, we
  still need to check <code>isWord</code> at the end.
</Intuition>

<InterviewTip>
  <code>to</code> and <code>top</code> can share letters. That is why the trie
  needs a separate word-end check instead of only checking the path.
</InterviewTip>

## TypeScript Solution

```ts
search(word: string): boolean {
  let node = this.root;

  for (const letter of word) {
    const nextNode = node.children.get(letter);

    if (!nextNode) {
      return false;
    }

    node = nextNode;
  }

  return node.isWord;
}
```
