<ProblemStatement title="Count how many words begin the same way">
  Given a trie and a prefix, return how many stored words start with that
  prefix.
</ProblemStatement>

## Example

```txt
stored words: to, top, toy, ten
count("to") -> 3
```

<Intuition title="First find the doorway, then count below it">
  The prefix path leads us to the node where that beginning ends. From there, we
  explore all the child paths below it and count every node marked with
  <code>isWord</code>.
</Intuition>

<InterviewTip>
  This is a two-part plan: follow the prefix, then do DFS to count full words in
  that smaller subtree.
</InterviewTip>

## TypeScript Solution

```ts
countWordsWithPrefix(prefix: string): number {
  let node = this.root;

  for (const letter of prefix) {
    const nextNode = node.children.get(letter);
    if (!nextNode) {
      return 0;
    }

    node = nextNode;
  }

  const dfs = (current: TrieNode): number => {
    let total = current.isWord ? 1 : 0;

    for (const child of current.children.values()) {
      total += dfs(child);
    }

    return total;
  };

  return dfs(node);
}
```
