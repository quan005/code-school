<ProblemStatement title="Find the biggest shared beginning">
  Given an array of strings, return the longest prefix shared by all of them.
</ProblemStatement>

## Example

```txt
words = ["flower", "flow", "flight"]
answer = "fl"
```

<Intuition title="Walk while everyone still agrees">
  Insert all the words into the trie. Then start at the root and keep walking
  down as long as there is only one next child and no word ends too soon. The
  moment the path splits, the shared prefix is over.
</Intuition>

<InterviewTip>
  Branching means disagreement. If one node has more than one child, the shared
  prefix stops there.
</InterviewTip>

## TypeScript Solution

```ts
function longestCommonPrefix(words: string[]): string {
  const trie = new Trie();

  for (const word of words) {
    trie.insert(word);
  }

  let node = trie.root;
  let prefix = "";

  while (node.children.size === 1 && !node.isWord) {
    const [letter, nextNode] = node.children.entries().next().value as [
      string,
      TrieNode,
    ];
    prefix += letter;
    node = nextNode;
  }

  return prefix;
}
```
