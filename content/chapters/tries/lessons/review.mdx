## Pattern Summary

- Insert: build missing child nodes and mark the end.
- Exact search: follow the path and check <code>isWord</code>.
- Prefix search: follow the path only.
- Wildcard search: try all children when a dot appears.
- Prefix count: find the prefix node, then count word ends below it.
- Longest common prefix: keep walking until the trie branches or a word ends.
