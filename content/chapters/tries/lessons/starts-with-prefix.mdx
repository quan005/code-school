<ProblemStatement title="Ask whether any word starts this way">
  Given a trie and a prefix, return <code>true</code> if at least one stored
  word starts with that prefix.
</ProblemStatement>

## Example

```txt
stored words: cat, car, cap
startsWith("ca") -> true
startsWith("cab") -> false
```

<Intuition title="For prefix search, the path matters more than the ending mark">
  We only care whether the letters of the prefix can be followed in the trie. We
  do not need <code>isWord</code> at the end because the prefix can be only the
  beginning of a longer word.
</Intuition>

<InterviewTip>
  Exact search asks, "Is the whole word here?" Prefix search asks, "Can I walk
  this beginning path?"
</InterviewTip>

## TypeScript Solution

```ts
startsWith(prefix: string): boolean {
  let node = this.root;

  for (const letter of prefix) {
    const nextNode = node.children.get(letter);

    if (!nextNode) {
      return false;
    }

    node = nextNode;
  }

  return true;
}
```
