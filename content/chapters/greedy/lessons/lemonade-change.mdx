<ProblemStatement title="Can you give the right change to every customer?">
  Each lemonade costs $5. Customers pay with $5, $10, or $20. Return
  <code>true</code> if you can give correct change to everyone in order.
</ProblemStatement>

## Example

```txt
bills = [5, 5, 5, 10, 20]
answer = true
```

<Intuition title="Some bills are more flexible than others">
  A $5 bill is very useful because both $10 and $20 customers may need it for
  change. So when a $20 customer comes, greedy prefers giving one $10 and one $5
  instead of three $5 bills when possible.
</Intuition>

<InterviewTip>
  Greedy often works when one local choice clearly protects future flexibility.
  Here, saving $5 bills helps future customers.
</InterviewTip>

## TypeScript Solution

```ts
function lemonadeChange(bills: number[]): boolean {
  let fives = 0;
  let tens = 0;

  for (const bill of bills) {
    if (bill === 5) {
      fives += 1;
    } else if (bill === 10) {
      if (fives === 0) {
        return false;
      }

      fives -= 1;
      tens += 1;
    } else {
      if (tens > 0 && fives > 0) {
        tens -= 1;
        fives -= 1;
      } else if (fives >= 3) {
        fives -= 3;
      } else {
        return false;
      }
    }
  }

  return true;
}
```
