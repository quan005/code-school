<ProblemStatement title="Take the maximum total profit with many buys and sells">
  Given stock prices by day, return the maximum profit when you may buy and sell
  many times but can hold only one stock at a time.
</ProblemStatement>

## Example

```txt
prices = [7, 1, 5, 3, 6, 4]
answer = 7
```

<Intuition title="Every positive rise is worth taking">
  If tomorrow’s price is higher than today’s, that gain can safely be added.
  Greedy says: take every day-to-day profit jump greater than zero.
</Intuition>

<InterviewTip>
  This greedy rule works because a long rise and all its little positive pieces
  add up to the same total gain.
</InterviewTip>

## TypeScript Solution

```ts
function maxProfit(prices: number[]): number {
  let profit = 0;

  for (let day = 1; day < prices.length; day += 1) {
    if (prices[day] > prices[day - 1]) {
      profit += prices[day] - prices[day - 1];
    }
  }

  return profit;
}
```
