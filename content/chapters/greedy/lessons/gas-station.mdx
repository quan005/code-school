<ProblemStatement title="Find a starting station that completes the full circle">
  Given gas at each station and the cost to travel to the next station, return
  the starting station index if the whole circle can be completed. Otherwise,
  return <code>-1</code>.
</ProblemStatement>

## Example

```txt
gas = [1, 2, 3, 4, 5]
cost = [3, 4, 5, 1, 2]
answer = 3
```

<Intuition title="First check if the whole trip is possible at all">
  If total gas is less than total cost, no start can ever work. If the whole
  trip is possible, greedy tracks a running tank. When that tank becomes
  negative, the current start fails, so we reset the start to the next station.
</Intuition>

<InterviewTip>
  The reset rule is the greedy heart of this problem: if one whole section fails
  from the current start, none of the starts inside that failed section can do
  better.
</InterviewTip>

## TypeScript Solution

```ts
function canCompleteCircuit(gas: number[], cost: number[]): number {
  let total = 0;
  let tank = 0;
  let start = 0;

  for (let index = 0; index < gas.length; index += 1) {
    const diff = gas[index] - cost[index];
    total += diff;
    tank += diff;

    if (tank < 0) {
      start = index + 1;
      tank = 0;
    }
  }

  return total < 0 ? -1 : start;
}
```
