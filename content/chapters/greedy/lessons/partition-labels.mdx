<ProblemStatement title="Split the string into as many safe parts as possible">
  Given a string <code>s</code>, split it into as many parts as possible so that
  each letter appears in at most one part. Return the sizes of the parts.
</ProblemStatement>

## Example

```txt
s = "ababcbacadefegdehijhklij"
answer = [9, 7, 8]
```

<Intuition title="Each seen letter tells how long the current part must stay open">
  First record the last position of every letter. Then scan from left to right.
  If a letter’s last position is farther away, the current partition must stay
  open until that point.
</Intuition>

<InterviewTip>
  Greedy cuts the partition at the earliest safe point, which is exactly when
  all letters inside it are finished.
</InterviewTip>

## TypeScript Solution

```ts
function partitionLabels(s: string): number[] {
  const last = new Map<string, number>();

  for (let index = 0; index < s.length; index += 1) {
    last.set(s[index], index);
  }

  const result: number[] = [];
  let start = 0;
  let end = 0;

  for (let index = 0; index < s.length; index += 1) {
    end = Math.max(end, last.get(s[index])!);

    if (index === end) {
      result.push(end - start + 1);
      start = index + 1;
    }
  }

  return result;
}
```
