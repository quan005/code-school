<ProblemStatement title="How tall is the tree?">
  Given the root of a binary tree, return its maximum depth. The maximum depth
  is the number of nodes along the longest path from the root down to a leaf.
</ProblemStatement>

## Example

```txt
        3
       / \
      9   20
         /  \
        15   7

answer = 3
```

<Intuition title="Each node asks its children how deep they are">
  At each node, we look at the left subtree depth and right subtree depth, keep
  the bigger one, and add one for the current node.
</Intuition>

<InterviewTip>
  Many tree-recursion problems follow the same shape: solve left, solve right,
  then combine the two answers.
</InterviewTip>

## TypeScript Solution

```ts
function maxDepth(root: TreeNode | null): number {
  if (root === null) {
    return 0;
  }

  const leftDepth = maxDepth(root.left);
  const rightDepth = maxDepth(root.right);

  return 1 + Math.max(leftDepth, rightDepth);
}
```
