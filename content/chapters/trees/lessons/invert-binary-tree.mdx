<ProblemStatement title="Flip the tree like a mirror">
  Given the root of a binary tree, invert the tree and return its root. To
  invert means swap the left and right children of every node.
</ProblemStatement>

## Example

```txt
        4
       / \
      2   7
     / \ / \
    1  3 6  9
```

becomes

```txt
        4
       / \
      7   2
     / \ / \
    9  6 3  1
```

<Intuition title="Do one local swap, then repeat in the subtrees">
  At every node, the local job is simple: swap left and right. Then do the same
  thing to the smaller trees below it.
</Intuition>

<InterviewTip>
  When the same small action should happen at every node, recursion is often a
  natural fit.
</InterviewTip>

## TypeScript Solution

```ts
function invertTree(root: TreeNode | null): TreeNode | null {
  if (root === null) {
    return null;
  }

  const temp = root.left;
  root.left = root.right;
  root.right = temp;

  invertTree(root.left);
  invertTree(root.right);

  return root;
}
```
