<ProblemStatement title="Do these two trees really match?">
  Given two binary trees, return <code>true</code> if they are the same. They
  are the same only if they have the same shape and the same values in matching
  places.
</ProblemStatement>

## Example

```txt
    1         1
   / \       / \
  2   3     2   3
```

<Intuition title="Compare matching nodes together">
  We compare both trees at the same time. If one side has a node and the other
  side does not, they are different. If both have nodes but the values do not
  match, they are different too.
</Intuition>

<InterviewTip>
  Same-tree problems are often really “compare left with left and right with
  right” recursion problems.
</InterviewTip>

## TypeScript Solution

```ts
function isSameTree(p: TreeNode | null, q: TreeNode | null): boolean {
  if (p === null && q === null) {
    return true;
  }

  if (p === null || q === null) {
    return false;
  }

  if (p.val !== q.val) {
    return false;
  }

  return isSameTree(p.left, q.left) && isSameTree(p.right, q.right);
}
```
