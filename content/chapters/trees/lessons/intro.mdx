<ProblemStatement title="A tree branches out from one starting point">
  A tree is a group of connected nodes that branches out from a top starting
  point called the root.
</ProblemStatement>

<Intuition title="Think about a family tree or folders on a computer">
  One top folder can contain smaller folders, and those can contain even more.
  That branching shape is tree thinking.
</Intuition>

## Helpful tree words

- <strong>root</strong>: the top starting node
- <strong>parent</strong>: a node above another node
- <strong>child</strong>: a node below another node
- <strong>leaf</strong>: a node with no children

## Two big ways to move through a tree

- <strong>DFS</strong>: go deep down one path first.
- <strong>BFS</strong>: go level by level.

## Lesson Summary

- A tree branches from a root.
- A node is one item in the tree.
- Leaves have no children.
- DFS goes deep first.
- BFS goes level by level.
