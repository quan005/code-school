<ProblemStatement title="Can all the classes be finished?">
  Given courses and prerequisite pairs, return <code>true</code> if you can
  finish all courses. Return <code>false</code> if a cycle blocks the schedule.
</ProblemStatement>

## Example

```txt
Math 1 -> Math 2 -> Math 3

possible
```

```txt
A -> B -> C -> A

not possible
```

<Intuition title="A cycle means you are stuck waiting forever">
  If course A needs B, B needs C, and C needs A, then there is no first class
  you can start. That loop is called a cycle.
</Intuition>

<InterviewTip>
  Dependency problems often mean directed graph thinking. Cycle detection is the
  key clue here.
</InterviewTip>

## TypeScript Solution

```ts
function canFinish(numCourses: number, prerequisites: number[][]): boolean {
  const graph = Array.from({ length: numCourses }, () => [] as number[]);

  for (const [course, prereq] of prerequisites) {
    graph[prereq].push(course);
  }

  const visiting = new Set<number>();
  const visited = new Set<number>();

  function dfs(course: number): boolean {
    if (visiting.has(course)) {
      return false;
    }

    if (visited.has(course)) {
      return true;
    }

    visiting.add(course);

    for (const next of graph[course]) {
      if (!dfs(next)) {
        return false;
      }
    }

    visiting.delete(course);
    visited.add(course);
    return true;
  }

  for (let course = 0; course < numCourses; course += 1) {
    if (!dfs(course)) {
      return false;
    }
  }

  return true;
}
```
