<ProblemStatement title="Make a full copy of the graph">
  Given a node in a connected graph, return a deep copy of the whole graph.
</ProblemStatement>

## Example Idea

If node <code>A</code> is connected to <code>B</code> and <code>C</code>, the
clone should create new nodes for <code>A</code>, <code>B</code>, and

<code>C</code>, then connect the new nodes in the same way.

<Intuition title="Use a notebook to match old nodes to new nodes">
  When graphs can loop, you must not create a fresh copy every time you see the
  same node again. Keep a map from old node to new node. If a node already has a
  clone, reuse that clone.
</Intuition>

<InterviewTip>
  Clone Graph is a graph-traversal problem plus a hash map memory problem.
</InterviewTip>

## TypeScript Solution

```ts
function cloneGraph(node: GraphNode | null): GraphNode | null {
  if (node === null) {
    return null;
  }

  const clones = new Map<GraphNode, GraphNode>();

  function dfs(current: GraphNode): GraphNode {
    if (clones.has(current)) {
      return clones.get(current)!;
    }

    const copy = new GraphNode(current.val);
    clones.set(current, copy);

    for (const neighbor of current.neighbors) {
      copy.neighbors.push(dfs(neighbor));
    }

    return copy;
  }

  return dfs(node);
}
```
