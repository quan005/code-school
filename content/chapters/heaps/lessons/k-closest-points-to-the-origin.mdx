<ProblemStatement title="Keep the closest points">
  You are given points like <code>[x, y]</code>. Return the <code>k</code>
  points closest to the origin <code>(0, 0)</code>.
</ProblemStatement>

## Example

```txt
points = [[1, 3], [-2, 2], [2, -1]], k = 2
answer = [[-2, 2], [2, -1]]
```

<Intuition title="Keep the k closest, remove the farthest of those best points">
  To compare closeness, we can use squared distance: <code>x*x + y*y</code>. We
  keep a max-heap of size <code>k</code> so the farthest of the current best
  points is easy to remove.
</Intuition>

<InterviewTip>
  We use squared distance instead of true distance because it compares points
  correctly without needing square roots.
</InterviewTip>

## TypeScript Solution

```ts
function kClosest(points: number[][], k: number): number[][] {
  const maxHeap = new Heap<[number, number[]]>((a, b) => a[0] > b[0]);

  for (const point of points) {
    const distance = point[0] * point[0] + point[1] * point[1];
    maxHeap.push([distance, point]);

    if (maxHeap.size() > k) {
      maxHeap.pop();
    }
  }

  const result: number[][] = [];

  while (maxHeap.size() > 0) {
    result.push(maxHeap.pop()![1]);
  }

  return result;
}
```
