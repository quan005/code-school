<ProblemStatement title="Reverse the 32 bits of the number">
  Given a 32-bit unsigned integer, reverse its bits and return the result.
</ProblemStatement>

## Small Example

```txt
1101 -> 1011
```

<Intuition title="Read from the right, build on the left">
  Look at the current last bit of the number. Add that bit to the result, then
  shift the result left to make room for the next bit. Shift the original number
  right to move the next bit into place.
</Intuition>

<InterviewTip>
  Reverse Bits is like reading the original number one tiny bit at a time and
  writing a new number in the opposite order.
</InterviewTip>

## TypeScript Solution

```ts
function reverseBits(n: number): number {
  let result = 0;

  for (let count = 0; count < 32; count += 1) {
    result = (result << 1) | (n & 1);
    n = n >>> 1;
  }

  return result >>> 0;
}
```
