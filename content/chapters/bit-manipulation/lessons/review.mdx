## Pattern Summary

- Single Number: XOR cancels equal pairs.
- Number of 1 Bits: check the last bit, then shift.
- Power of Two: one 1 bit means n & (n - 1) becomes zero.
- Missing Number: XOR the full range with the array.
- Counting Bits: use a shifted smaller answer plus the last bit.
- Reverse Bits: build the answer one bit at a time.
