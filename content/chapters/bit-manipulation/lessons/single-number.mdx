<ProblemStatement title="Find the number that appears only once">
  Given an array where every number appears twice except for one number that
  appears once, return the number that appears once.
</ProblemStatement>

## Example

```txt
nums = [4, 1, 2, 1, 2]
answer = 4
```

<Intuition title="XOR makes matching pairs disappear">
  XOR has a powerful rule: <code>x ^ x = 0</code> and <code>x ^ 0 = x</code>. So
  when we XOR every number in the array, the pairs cancel out and the single
  number stays.
</Intuition>

<InterviewTip>
  This is one of the most famous bit tricks. If equal pairs should cancel, think
  about XOR.
</InterviewTip>

## TypeScript Solution

```ts
function singleNumber(nums: number[]): number {
  let result = 0;

  for (const num of nums) {
    result ^= num;
  }

  return result;
}
```
