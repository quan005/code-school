<ProblemStatement title="Count the on switches in the number">
  Given a positive integer <code>n</code>, return how many <code>1</code> bits
  are in its binary form.
</ProblemStatement>

## Example

```txt
11 = 1011
answer = 3
```

<Intuition title="Check the last bit, then shift">
  Use <code>n & 1</code> to check whether the current last bit is 1. Then shift
  right so the next bit moves into the last spot.
</Intuition>

<InterviewTip>
  <code>& 1</code> asks, “Is the last bit on?” A right shift moves us to the
  next bit.
</InterviewTip>

## TypeScript Solution

```ts
function hammingWeight(n: number): number {
  let count = 0;

  while (n !== 0) {
    count += n & 1;
    n = n >>> 1;
  }

  return count;
}
```
