<ProblemStatement title="Decide whether the number is a power of two">
  Given an integer <code>n</code>, return <code>true</code> if it is a power of
  two. Otherwise, return <code>false</code>.
</ProblemStatement>

## Example

```txt
8 = 1000 -> true
10 = 1010 -> false
```

<Intuition title="Powers of two have one lonely 1 bit">
  In binary, powers of two look like 1, 10, 100, 1000, and so on. They have
  exactly one <code>1</code> bit.
</Intuition>

<InterviewTip>
  The trick <code>n & (n - 1)</code> removes the lowest 1 bit. If the result is
  zero and the number was positive, that means there was only one 1 bit.
</InterviewTip>

## TypeScript Solution

```ts
function isPowerOfTwo(n: number): boolean {
  if (n <= 0) {
    return false;
  }

  return (n & (n - 1)) === 0;
}
```
