<ProblemStatement title="Return the bit counts for every number from 0 to n">
  Given an integer <code>n</code>, return an array where
  <code>answer[i]</code> is the number of <code>1</code> bits in the binary form
  of <code>i</code>.
</ProblemStatement>

## Example

```txt
n = 5
answer = [0, 1, 1, 2, 1, 2]
```

<Intuition title="A number is its shifted smaller self plus its last bit">
  If we shift <code>i</code> right by 1, we remove the last bit. So the bit
  count for <code>i</code> is the count for <code>i >> 1</code> plus
  <code>(i & 1)</code>, which tells whether the removed last bit was 1.
</Intuition>

<InterviewTip>
  This lesson mixes bit thinking with a tiny dynamic-programming idea because
  earlier answers help build later ones.
</InterviewTip>

## TypeScript Solution

```ts
function countBits(n: number): number[] {
  const result = new Array(n + 1).fill(0);

  for (let value = 1; value <= n; value += 1) {
    result[value] = result[value >> 1] + (value & 1);
  }

  return result;
}
```
