<ProblemStatement title="Bit manipulation works with the tiny 0-and-1 pieces inside numbers">
  Inside a computer, numbers are stored using bits. A bit can only be
  <code>0</code> or <code>1</code>.
</ProblemStatement>

<Intuition title="Think about light switches">
  A bit is like a tiny light switch. <code>0</code> means off, and
  <code>1</code> means on. When several bits sit together, they can represent a
  number.
</Intuition>

<InterviewTip>
  In binary, the place values are 1, 2, 4, 8, 16, and so on. For example,
  <code>1011</code> means 8 + 2 + 1, which equals 11.
</InterviewTip>

## Helpful bit ideas

- <strong>bit</strong>: one tiny 0 or 1
- <strong>binary</strong>: writing numbers with bits
- <strong>AND</strong>: 1 only when both bits are 1
- <strong>XOR</strong>: 1 when the bits are different
- <strong>shift</strong>: move bits left or right

## Strong bit clues

- a number appears once while others repeat
- count how many 1 bits are in a number
- check whether a number is a power of two
- compare a full expected set with one missing value
- build a number one bit at a time

## Lesson Summary

- A bit is one tiny 0 or 1.
- Binary writes numbers with bit place values like 1, 2, 4, and 8.
- XOR, AND, and shifts are common bit tools.
- Bit tricks work best when the problem matches a binary pattern.
- Bits can help solve some number problems very neatly.
