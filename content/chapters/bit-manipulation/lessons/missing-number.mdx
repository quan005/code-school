<ProblemStatement title="Find the one missing value from 0 through n">
  Given an array containing <code>n</code> distinct numbers from the range
  <code>0</code> to <code>n</code>, return the one number that is missing.
</ProblemStatement>

## Example

```txt
nums = [3, 0, 1]
answer = 2
```

<Intuition title="Compare what should exist with what actually exists">
  XOR together every index from <code>0</code> to <code>n</code> and every value
  in the array. The matches cancel out, and the missing number is the one value
  left over.
</Intuition>

<InterviewTip>
  XOR is great when you have a full expected set and one incomplete actual set.
</InterviewTip>

## TypeScript Solution

```ts
function missingNumber(nums: number[]): number {
  let result = nums.length;

  for (let index = 0; index < nums.length; index += 1) {
    result ^= index;
    result ^= nums[index];
  }

  return result;
}
```
