<ProblemStatement title="A window is a moving section">
  A sliding window is a part of an array or string that we keep track of while
  it moves through the data.
</ProblemStatement>

<Intuition title="Picture a frame sliding across a row">
  Imagine holding a picture frame over a long row of objects. The frame shows
  only one section at a time. When the frame slides, one thing leaves and
  another thing enters.
</Intuition>

<InterviewTip>
  Sliding windows are usually a clue when the problem talks about contiguous
  sections, subarrays, substrings, longest, shortest, or a fixed size
  <code>k</code>.
</InterviewTip>

<ComplexityCard
  time="Often O(n)"
  space="Often O(1) to O(k)"
  notes="The main win is reusing work from the current window instead of rebuilding every section from scratch."
/>

## Two main styles

- <strong>Fixed-size window</strong>: the width stays the same.
- <strong>Dynamic window</strong>: the width grows and shrinks as needed.

## Helpful variables

- <code>left</code> marks the start of the window.
- <code>right</code> marks the end of the window.

## Lesson Summary

- A window is a moving section.
- Fixed windows keep the same width.
- Dynamic windows grow and shrink.
- Left and right pointers describe the current section.
