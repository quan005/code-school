<ProblemStatement title="Two window styles to remember">
  Sliding windows usually come in two styles: fixed-size windows and dynamic
  windows.
</ProblemStatement>

<Intuition title="Fast recap by problem">
  Maximum Sum and Average use fixed-size windows. Smallest Subarray, Longest
  Substring Without Repeats, and Character Replacement use dynamic windows.
  Permutation in a String is fixed-size plus a map of counts.
</Intuition>
