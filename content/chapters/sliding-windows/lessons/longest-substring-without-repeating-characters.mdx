<ProblemStatement title="Find the longest substring with no repeated characters">
  Given a string <code>s</code>, return the length of the longest substring that
  has no repeating characters.
</ProblemStatement>

## Examples

```txt
s = "abcabcbb" -> 3
s = "bbbbb" -> 1
```

<Intuition title="Keep the window unique">
  A set can remember which characters are inside the current window. When a
  repeat appears, shrink from the left until the repeat is gone.
</Intuition>

<InterviewTip>
  Use <code>while</code> when shrinking. One left move may not be enough to
  remove the repeated character.
</InterviewTip>

## TypeScript Solution

```ts
function lengthOfLongestSubstring(s: string): number {
  let left = 0;
  let best = 0;
  const seen = new Set<string>();

  for (let right = 0; right < s.length; right++) {
    while (seen.has(s[right])) {
      seen.delete(s[left]);
      left++;
    }

    seen.add(s[right]);
    best = Math.max(best, right - left + 1);
  }

  return best;
}
```
