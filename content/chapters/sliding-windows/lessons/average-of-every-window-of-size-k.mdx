<ProblemStatement title="Return the average of each fixed-size window">
  Given an array of numbers <code>nums</code> and an integer <code>k</code>,
  return an array containing the average of every contiguous subarray of size{" "}
  <code>k</code>.
</ProblemStatement>

## Example

```txt
nums = [1, 3, 2, 6, -1, 4, 1, 8, 2], k = 5
output = [2.2, 2.8, 2.4, 3.6, 2.8]
```

<Intuition title="Same window trick, new output">
  This lesson uses the same running-sum idea as Maximum Sum of Size K. The
  difference is that we save each window’s average instead of keeping only the
  best sum.
</Intuition>

<InterviewTip>
  Once the window is big enough, record the answer, then slide by removing the
  leftmost value.
</InterviewTip>

## TypeScript Solution

```ts
function averagesOfSubarrays(nums: number[], k: number): number[] {
  const result: number[] = [];

  if (k <= 0 || k > nums.length) {
    return result;
  }

  let windowSum = 0;
  let left = 0;

  for (let right = 0; right < nums.length; right++) {
    windowSum += nums[right];

    if (right >= k - 1) {
      result.push(windowSum / k);
      windowSum -= nums[left];
      left++;
    }
  }

  return result;
}
```
