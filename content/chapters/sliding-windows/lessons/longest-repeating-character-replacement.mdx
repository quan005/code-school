<ProblemStatement title="Find the longest fixable substring">
  You are given a string <code>s</code> and an integer <code>k</code>. You may
  change at most <code>k</code> characters so that all characters in a substring
  become the same. Return the length of the longest substring you can make this
  way.
</ProblemStatement>

## Example

```txt
s = "AABABBA", k = 1 -> 4
```

<Intuition title="A window is valid if it can be fixed">
  Suppose the current window has size 5 and the most common letter appears 4
  times. Then only 1 change is needed. The rule is:
</Intuition>

```txt
window size - most common letter count <= k
```

<InterviewTip>
  This lesson is advanced because the rule is not about exact matching. The
  window is okay as long as it is still fixable.
</InterviewTip>

## TypeScript Solution

```ts
function characterReplacement(s: string, k: number): number {
  const counts = new Map<string, number>();
  let left = 0;
  let maxCount = 0;
  let best = 0;

  for (let right = 0; right < s.length; right++) {
    const ch = s[right];
    counts.set(ch, (counts.get(ch) ?? 0) + 1);
    maxCount = Math.max(maxCount, counts.get(ch)!);

    while (right - left + 1 - maxCount > k) {
      const leftChar = s[left];
      counts.set(leftChar, counts.get(leftChar)! - 1);
      left++;
    }

    best = Math.max(best, right - left + 1);
  }

  return best;
}
```

<HintDrawer title="What does maxCount tell us?">
  MaxCount tells us how many characters are already the most common letter in
  the current window. The rest are the ones we would need to change.
</HintDrawer>
