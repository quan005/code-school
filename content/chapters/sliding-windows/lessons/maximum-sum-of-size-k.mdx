<ProblemStatement title="Find the biggest sum in any window of size k">
  Given an array of positive integers <code>nums</code> and a number{" "}
  <code>k</code>, return the maximum sum of any contiguous subarray of size{" "}
  <code>k</code>.
</ProblemStatement>

## Example

```txt
nums = [2, 1, 5, 1, 3, 2], k = 3 -> 9
```

<Intuition title="Reuse the current sum">
  Instead of adding every group of 3 from scratch, build the first window once,
  then slide right by removing the old left value and adding the new right
  value.
</Intuition>

<InterviewTip>
  In fixed-size windows, ask yourself: what leaves when the window slides, and
  what enters?
</InterviewTip>

## TypeScript Solution

```ts
function maxSumSubarrayOfSizeK(nums: number[], k: number): number {
  if (k <= 0 || k > nums.length) {
    return 0;
  }

  let windowSum = 0;

  for (let i = 0; i < k; i++) {
    windowSum += nums[i];
  }

  let maxSum = windowSum;

  for (let right = k; right < nums.length; right++) {
    windowSum += nums[right];
    windowSum -= nums[right - k];
    maxSum = Math.max(maxSum, windowSum);
  }

  return maxSum;
}
```
