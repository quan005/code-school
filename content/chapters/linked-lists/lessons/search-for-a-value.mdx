<ProblemStatement title="Check whether the target is in the chain">
  Given the <code>head</code> of a linked list and a target value, return{" "}
  <code>true</code> if the value is in the list. Otherwise, return{" "}
  <code>false</code>.
</ProblemStatement>

## Examples

```txt
2 -> 5 -> 9 -> null, target = 5 -> true
2 -> 5 -> 9 -> null, target = 7 -> false
```

<Intuition title="Search feels like checking train cars">
  You cannot skip to the middle. You must check one node, then the next, then
  the next until you either find the value or run out of list.
</Intuition>

<InterviewTip>
  A linked list search looks like array search, but the movement is different:
  arrays use indexes, linked lists use pointers.
</InterviewTip>

## TypeScript Solution

```ts
function searchList(head: ListNode | null, target: number): boolean {
  let current = head;

  while (current !== null) {
    if (current.val === target) {
      return true;
    }

    current = current.next;
  }

  return false;
}
```

<HintDrawer title="What tells us the answer is false?">
  Reaching null means the search walked through the whole chain and never found
  the target.
</HintDrawer>
