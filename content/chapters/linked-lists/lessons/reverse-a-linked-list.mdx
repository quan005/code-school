<ProblemStatement title="Flip the chain around">
  Given the <code>head</code> of a linked list, reverse the list and return the
  new head.
</ProblemStatement>

## Example

```txt
1 -> 2 -> 3 -> null
result -> 3 -> 2 -> 1 -> null
```

<Intuition title="Reverse means flipping arrows">
  To reverse safely, we need three pointers: <code>prev</code>,{" "}
  <code>current</code>, and <code>nextNode</code>. We save the rest of the list
  before flipping the current arrow.
</Intuition>

<InterviewTip>
  The key danger is losing the rest of the chain. That is why{" "}
  <code>nextNode</code> must be saved before changing <code>current.next</code>.
</InterviewTip>

## TypeScript Solution

```ts
function reverseList(head: ListNode | null): ListNode | null {
  let prev: ListNode | null = null;
  let current: ListNode | null = head;

  while (current !== null) {
    const nextNode = current.next;
    current.next = prev;
    prev = current;
    current = nextNode;
  }

  return prev;
}
```
