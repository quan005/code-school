<ProblemStatement title="Remove the first matching node">
  Given the <code>head</code> of a linked list and a target value, delete the
  first node whose value equals the target and return the updated head.
</ProblemStatement>

## Example

```txt
3 -> 7 -> 9 -> null
delete 7
result -> 3 -> 9 -> null
```

<Intuition title="Deletion means reconnecting">
  We usually do not erase the node directly. Instead, we make the node before it
  skip over it by changing one pointer.
</Intuition>

<InterviewTip>
  The head is a special case. If the head holds the target, the answer becomes
  head.next.
</InterviewTip>

## TypeScript Solution

```ts
function deleteByValue(head: ListNode | null, target: number): ListNode | null {
  if (head === null) {
    return null;
  }

  if (head.val === target) {
    return head.next;
  }

  let current = head;

  while (current.next !== null) {
    if (current.next.val === target) {
      current.next = current.next.next;
      return head;
    }

    current = current.next;
  }

  return head;
}
```

<HintDrawer title="What does skipping mean?">
  If current points to 3 and current.next points to 7, then setting current.next
  to 7.next makes the chain jump over 7.
</HintDrawer>
