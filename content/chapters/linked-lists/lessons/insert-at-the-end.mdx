<ProblemStatement title="Add a new node at the tail">
  Given the <code>head</code> of a linked list and a new value, insert a new
  node with that value at the end of the list and return the head.
</ProblemStatement>

## Example

```txt
1 -> 4 -> null
insert 9
result -> 1 -> 4 -> 9 -> null
```

<Intuition title="Find the tail first">
  The tail is the node whose <code>next</code> is <code>null</code>. Once you
  find it, connect its <code>next</code> to the new node.
</Intuition>

<InterviewTip>
  The empty-list case matters. If the head is null, the new node becomes the
  whole list.
</InterviewTip>

## TypeScript Solution

```ts
function insertAtEnd(head: ListNode | null, value: number): ListNode {
  const newNode = new ListNode(value);

  if (head === null) {
    return newNode;
  }

  let current = head;

  while (current.next !== null) {
    current = current.next;
  }

  current.next = newNode;
  return head;
}
```
