<ProblemStatement title="Visit every node from start to end">
  Given the <code>head</code> of a linked list, print or collect all the values
  from start to end.
</ProblemStatement>

## Example

```txt
head -> 4 -> 8 -> 2 -> null
output -> [4, 8, 2]
```

<Intuition title="Follow the chain">
  Traversing means starting at the head and following <code>next</code> until
  there is no next node left.
</Intuition>

<InterviewTip>
  The most important loop rule is: while the current node is not null, use it,
  then move forward.
</InterviewTip>

## Plan

1. Start with <code>current = head</code>.
2. While <code>current !== null</code>, use <code>current.val</code>.
3. Move to <code>current.next</code>.
4. Stop when <code>current</code> becomes <code>null</code>.

## TypeScript Solution

```ts
function traverseList(head: ListNode | null): number[] {
  const values: number[] = [];
  let current = head;

  while (current !== null) {
    values.push(current.val);
    current = current.next;
  }

  return values;
}
```
