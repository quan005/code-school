<ProblemStatement title="Place the new interval in the right spot">
  You are given a sorted, non-overlapping list of intervals and one new
  interval. Insert the new interval and merge if needed.
</ProblemStatement>

## Example

```txt
intervals = [[1, 3], [6, 9]]
newInterval = [2, 5]
answer = [[1, 5], [6, 9]]
```

<Intuition title="This problem naturally breaks into three parts">
  As we scan the sorted intervals, each one is either before the new interval,
  overlapping with it, or after it. That gives us a clean one-pass plan.
</Intuition>

<InterviewTip>
  Already-sorted, non-overlapping intervals make insertion much easier because
  the overlap section appears as one block.
</InterviewTip>

## TypeScript Solution

```ts
function insertInterval(
  intervals: [number, number][],
  newInterval: [number, number],
): [number, number][] {
  const result: [number, number][] = [];
  let i = 0;

  while (i < intervals.length && intervals[i][1] < newInterval[0]) {
    result.push(intervals[i]);
    i++;
  }

  while (i < intervals.length && intervals[i][0] <= newInterval[1]) {
    newInterval[0] = Math.min(newInterval[0], intervals[i][0]);
    newInterval[1] = Math.max(newInterval[1], intervals[i][1]);
    i++;
  }

  result.push(newInterval);

  while (i < intervals.length) {
    result.push(intervals[i]);
    i++;
  }

  return result;
}
```
