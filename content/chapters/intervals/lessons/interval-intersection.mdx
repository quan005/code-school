<ProblemStatement title="Find the shared parts of two interval lists">
  You are given two sorted interval lists. Return all intersections between
  them.
</ProblemStatement>

## Example

```txt
A = [[0, 2], [5, 10], [13, 23], [24, 25]]
B = [[1, 5], [8, 12], [15, 24], [25, 26]]

answer = [[1, 2], [5, 5], [8, 10], [15, 23], [24, 24], [25, 25]]
```

<Intuition title="The shared part uses the bigger start and smaller end">
  When two intervals overlap, the intersection starts at the later start and
  ends at the earlier end. Then we move forward in the list whose interval ends
  first.
</Intuition>

<InterviewTip>
  This is an interval problem that behaves like two pointers. Each list gives
  one current interval to compare.
</InterviewTip>

## TypeScript Solution

```ts
function intervalIntersection(
  A: [number, number][],
  B: [number, number][],
): [number, number][] {
  const result: [number, number][] = [];
  let i = 0;
  let j = 0;

  while (i < A.length && j < B.length) {
    const start = Math.max(A[i][0], B[j][0]);
    const end = Math.min(A[i][1], B[j][1]);

    if (start <= end) {
      result.push([start, end]);
    }

    if (A[i][1] < B[j][1]) {
      i++;
    } else {
      j++;
    }
  }

  return result;
}
```

<HintDrawer title="Shared middle only">
  The overlap is only the part both intervals cover, so use the later start and
  earlier end.
