<ProblemStatement title="Combine ranges that share space">
  Given a list of intervals, merge all overlapping intervals and return the
  merged result.
</ProblemStatement>

## Example

```txt
[[1, 3], [2, 6], [8, 10], [15, 18]]
becomes
[[1, 6], [8, 10], [15, 18]]
```

<Intuition title="Keep one current interval and grow it when needed">
  After sorting by start time, any intervals that overlap will appear next to
  each other. That lets us keep one current interval and either extend it or
  save it.
</Intuition>

<InterviewTip>
  Sorting is the key first move. It places possible overlap neighbors beside
  each other.
</InterviewTip>

## TypeScript Solution

```ts
function mergeIntervals(intervals: [number, number][]): [number, number][] {
  if (intervals.length <= 1) {
    return intervals;
  }

  intervals.sort((a, b) => a[0] - b[0]);

  const result: [number, number][] = [];
  let [start, end] = intervals[0];

  for (let i = 1; i < intervals.length; i++) {
    const [nextStart, nextEnd] = intervals[i];

    if (nextStart <= end) {
      end = Math.max(end, nextEnd);
    } else {
      result.push([start, end]);
      [start, end] = [nextStart, nextEnd];
    }
  }

  result.push([start, end]);
  return result;
}
```

<HintDrawer title="Ask one question each time">
  Does the next interval start before or at the current end? If yes, merge. If
  no, save the current interval and start a new one.
</HintDrawer>
