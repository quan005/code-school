<ProblemStatement title="Do any meetings collide?">
  You are given meeting intervals. Return <code>true</code> if one person can
  attend every meeting, and <code>false</code> if any meetings overlap.
</ProblemStatement>

## Example

```txt
[[0, 30], [15, 20]]
answer = false
```

<Intuition title="Any conflict must appear between neighbors after sorting">
  After sorting by start time, we only need to compare each meeting with the one
  right before it. If the next one starts too early, there is a conflict.
</Intuition>

<InterviewTip>
  This is a simpler interval problem because the answer is just yes or no. We do
  not need to build merged ranges.
</InterviewTip>

## TypeScript Solution

```ts
function canAttendAllMeetings(intervals: [number, number][]): boolean {
  intervals.sort((a, b) => a[0] - b[0]);

  for (let i = 1; i < intervals.length; i++) {
    if (intervals[i][0] < intervals[i - 1][1]) {
      return false;
    }
  }

  return true;
}
```
