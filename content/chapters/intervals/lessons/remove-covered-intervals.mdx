<ProblemStatement title="Remove intervals hidden inside bigger ones">
  Given a list of intervals, remove intervals that are completely covered by
  another interval. Return the number of intervals left.
</ProblemStatement>

## Example

```txt
[[1, 4], [3, 6], [2, 8]]
answer = 2
```

<Intuition title="A covered interval ends no farther than a stronger one before it">
  We sort by start time, and if starts tie, we put the bigger end first. Then we
  walk left to right and track the farthest end seen so far. Any interval that
  does not go farther is covered.
</Intuition>

<InterviewTip>
  The tie-break rule matters here. If starts tie, the longer interval must come
  first so it can cover the shorter one.
</InterviewTip>

## TypeScript Solution

```ts
function removeCoveredIntervals(intervals: [number, number][]): number {
  intervals.sort((a, b) => {
    if (a[0] === b[0]) {
      return b[1] - a[1];
    }
    return a[0] - b[0];
  });

  let count = 0;
  let farthestEnd = -Infinity;

  for (const [, end] of intervals) {
    if (end > farthestEnd) {
      count++;
      farthestEnd = end;
    }
  }

  return count;
}
```
